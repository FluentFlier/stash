#      - Recommended: LTS version (20.x or later)

#
#   2. Verify installation:
#      - Open terminal/command prompt
#      - Run: node --version
#      - Run: npm --version
#      - Both should show version numbers
#
#   3. Install Overshoot SDK:
#      - Run: npm install -g @overshoot/sdk
#      - Or from this directory: npm install @overshoot/sdk
#
#   4. The Python code will automatically call the Node.js SDK via subprocess
#
